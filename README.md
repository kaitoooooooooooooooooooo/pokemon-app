# 🔴 Pokemon App

> A full-stack Pokémon management application — Flutter mobile client and Node.js REST API with Swagger documentation.

## 🛠️ Tech Stack

- **Client** : Flutter / Dart
- **Server** : Node.js / Express
- **API Documentation** : Swagger
- **Database** : JSON

## 📁 Project Structure
```
POKEMON/
├── clientPokemon/        # Flutter mobile application
├── serverPokemon/        # Node.js REST API + Swagger
└── pokemon.pokemons.json # JSON database
```

## ⚙️ Installation

### Server
```bash
cd serverPokemon
npm install
npm start
```

### Client
```bash
cd clientPokemon
flutter pub get
flutter run
```

## 📖 API Documentation

Once the server is running, the Swagger documentation is available at :
```
http://localhost:PORT/api-docs
```
